Introduction

One of the more overlooked parts of designing a website is the UX design. A poorly designed UX can be hard or even impossible to navigate.

Bad Examples

Some websites are a bit more difficult to navigate due to some reason, be it lack of a efficient user flow, an inability to backtrack, or something else.

Navigation Design

Navigation Design is a subset of UX that deals with the flow of the user experience. It asks and answers the question of “How does the end user get here?” and “How does the user get back?”
